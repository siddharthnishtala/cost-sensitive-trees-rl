Installation Notes

The codebase uses slightly modified versions of scikit-learn and stable-baselines3 libraries. 
Building scikit-learn from source requires other dependencies. Please follow instructions
from the below link to setup before installing these libraries.

scikit-learn: https://scikit-learn.org/dev/developers/advanced_installation.html#building-from-source

Once this is setup, the libraries and all other dependencies can be installed using:

pip install -r requirements.txt

The code was written and tested on Python 3.7.9.
